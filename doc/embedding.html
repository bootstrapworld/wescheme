2 Parameters

The openEditor supports several parameters that are applicable in embedded mode.

  • publicId: the public id of the program to be embedded.

  • hideHeader: hides the header. Set to ’true’ to hide.

  • hideFooter: hides the footer. Set to ’true’ to hide.

  • hideDefinitions: hides the definitions pane

  • autorun: runs the definitions window if set to ’true’

  • warnOnExit: set this to ’false’ so the editor doesn’t yell when you exit the page.

  • interactionsText: if defined, its value will be used for the interactions pane.
