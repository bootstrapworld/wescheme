NOTE: updates to these files must be MANUALLY compiled, and added to the war/js/mzscheme-vm/collects/bootstrapXYZ/ folder

1) Update the file (be sure to test it in WeScheme first!)
2) Navigate to www.wescheme.org/js/compiler/
3) Copy-and-paste the source into the text box, and click “Download source as teachpack“
4) Enter the module name (e.g. - bootstrap2015/bootstrap-teachpack) and click OK
5) Save the resulting JS file into the appropriate folder inside war/js/mzscheme-vm/collects/
